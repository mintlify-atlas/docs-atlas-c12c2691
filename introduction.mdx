# Welcome to PARKINMX

PARKINMX is a comprehensive smart parking management platform designed to revolutionize how you find, reserve, and pay for parking spaces in Mexico. Built with React Native and Expo, the app delivers a seamless cross-platform experience on iOS, Android, and Web.

## What is PARKINMX?

PARKINMX transforms the traditional parking experience into a modern, digital-first solution. Users can view real-time parking availability, make reservations in advance, manage their vehicles and payment methods, and access their parking spots using secure QR codes - all from their mobile device or web browser.

The platform combines:
- **Real-time parking availability** with interactive maps
- **Advance reservation system** with time-slot booking
- **Digital wallet** with multiple payment options
- **QR-based access control** for contactless entry/exit
- **Vehicle management** for multiple cars
- **Community features** including friends and rankings

## Technology Stack

PARKINMX is built on modern, reliable technologies:

- **React Native 0.81.5** - Cross-platform mobile framework
- **Expo SDK 54** - Development platform with native capabilities
- **Firebase** - Backend services including authentication, Firestore database, and cloud functions
- **TypeScript** - Type-safe development
- **Expo Router 6** - File-based navigation system

<Note>
  PARKINMX uses Expo's new architecture with React Compiler and typed routes for enhanced performance and developer experience.
</Note>

## Security & Privacy

Your security is paramount:

- **Biometric authentication** using fingerprint or face recognition
- **Secure storage** with Expo SecureStore for sensitive data
- **PIN verification** for critical actions like making reservations
- **Email verification codes** for account recovery and transaction confirmation
- **Firebase Authentication** with industry-standard security protocols
