# Installing PARKINMX

This comprehensive guide covers installation on all supported platforms: iOS, Android, and Web. Choose your platform below for detailed instructions.

## System Requirements

Before installing, ensure your device meets these minimum requirements:

<Tabs>
  <Tab title="iOS">
    ### iOS Requirements
    
    - **iOS Version**: 13.0 or later
    - **Devices**: iPhone, iPad, iPod touch
    - **Storage**: Minimum 100 MB free space
    - **Internet**: WiFi or cellular data connection
    - **Optional**: Touch ID or Face ID for biometric login
    
    #### Compatible Devices
    - iPhone 6s and newer
    - iPad Air 2 and newer
    - iPad mini 4 and newer
    - iPad Pro (all models)
    - iPod touch (7th generation)
    
    <Note>
      PARKINMX supports both portrait and landscape orientations on iPad.
    </Note>
  </Tab>
  
  <Tab title="Android">
    ### Android Requirements
    
    - **Android Version**: 5.0 (Lollipop) or later
    - **Storage**: Minimum 100 MB free space
    - **RAM**: Minimum 2 GB recommended
    - **Internet**: WiFi or mobile data connection
    - **Optional**: Fingerprint sensor for biometric login
    
    #### Permissions Required
    The app requests these permissions:
    - `USE_BIOMETRIC` - For fingerprint/face authentication
    - `USE_FINGERPRINT` - Legacy fingerprint support
    - `CAMERA` - For QR code scanning (granted at runtime)
    - `INTERNET` - For app functionality
    
    <Tip>
      PARKINMX is optimized for edge-to-edge displays and supports Android's predictive back gesture.
    </Tip>
  </Tab>
  
  <Tab title="Web">
    ### Web Requirements
    
    - **Browsers**: Chrome 90+, Safari 14+, Firefox 88+, Edge 90+
    - **Internet**: Stable broadband or mobile connection
    - **Screen**: Minimum 320px width (mobile responsive)
    - **JavaScript**: Must be enabled
    
    #### Recommended Specifications
    - Modern browser with ES6+ support
    - 2 GB RAM or more
    - Stable connection (3G minimum, 4G/WiFi recommended)
    
    <Note>
      The web version is a Progressive Web App (PWA) and can be installed to your home screen for a native-like experience.
    </Note>
  </Tab>
</Tabs>

## Platform Installation

<Tabs>
  <Tab title="iOS">
    ## Installing on iOS
    
    <Steps>
      <Step title="Open App Store">
        1. Locate and tap the **App Store** icon on your home screen
        2. The App Store app is pre-installed on all iOS devices
        
        <Tip>
          Make sure you're signed in with your Apple ID before downloading apps.
        </Tip>
      </Step>
      
      <Step title="Search for PARKINMX">
        1. Tap the **Search** tab at the bottom of the screen
        2. Type **"PARKINMX"** in the search bar
        3. Tap **Search** on the keyboard
        4. Look for the PARKINMX app with the yellow and black icon
        
        <Note>
          The app icon features the PARKINMX logo on a light blue background (`#E6F4FE`).
        </Note>
      </Step>
      
      <Step title="Download the App">
        1. Tap the **Get** button next to PARKINMX
        2. Authenticate with Face ID, Touch ID, or Apple ID password
        3. The button changes to show download progress
        4. Wait for installation to complete (typically 30-60 seconds)
        
        The app will automatically appear on your home screen when installation is complete.
      </Step>
      
      <Step title="Launch and Set Up">
        1. Tap the **PARKINMX** icon on your home screen
        2. You'll see the splash screen with the PARKINMX logo
        3. After 2 seconds, you'll be taken to the login screen
        4. Choose **"Regístrate"** to create a new account or **"ENTRAR"** to log in
        
        ### First Launch Permissions
        
        On first launch, the app may request:
        
        - **Notifications**: Tap "Allow" to receive reservation updates and reminders
        - **Camera** (when needed): Required for QR code scanning at parking facilities
        - **Biometric Authentication**: Set up Touch ID or Face ID for quick login
        
        <Warning>
          Grant camera permission when prompted - it's essential for scanning parking QR codes.
        </Warning>
      </Step>
      
      <Step title="Configure Biometric Login (Optional)">
        Enable Face ID or Touch ID for faster authentication:
        
        1. After first login, tap **"Enable Biometric Login"** when prompted
        2. Follow the on-screen instructions
        3. Your biometric data is stored securely on-device only
        
        **Supported methods:**
        - **Face ID** - iPhone X and newer, iPad Pro (2018+)
        - **Touch ID** - iPhone 5s through iPhone 8, iPad with Home button
      </Step>
    </Steps>

    ## Troubleshooting iOS Installation
    
    <Accordion title="App Store says 'Cannot Download'">
      **Solutions:**
      1. Check your internet connection (try WiFi instead of cellular)
      2. Ensure you have enough storage space (Settings → General → iPhone Storage)
      3. Sign out and back into App Store (Settings → Your Name → Media & Purchases)
      4. Restart your device
      5. Update iOS to the latest version (Settings → General → Software Update)
    </Accordion>
    
    <Accordion title="Face ID / Touch ID not working">
      **Check these settings:**
      1. Go to Settings → Face ID & Passcode (or Touch ID & Passcode)
      2. Ensure PARKINMX is listed and enabled
      3. Try disabling and re-enabling biometric login in the app
      4. Make sure your device's biometric sensor is clean and unobstructed
    </Accordion>
    
    <Accordion title="App crashes on launch">
      **Try these fixes:**
      1. Force close the app (swipe up from bottom or double-click Home)
      2. Restart your device
      3. Delete and reinstall the app
      4. Check if iOS update is available
      5. Report the issue via GitHub if problem persists
    </Accordion>
    
    <Accordion title="Camera permission denied">
      **To grant camera access:**
      1. Open Settings → Privacy & Security → Camera
      2. Find PARKINMX in the list
      3. Toggle the switch to ON (green)
      4. Relaunch PARKINMX
    </Accordion>

    ## Installing on Android
    
    <Steps>
      <Step title="Open Google Play Store">
        1. Locate and tap the **Google Play Store** icon in your app drawer
        2. The Play Store is pre-installed on most Android devices
        3. Ensure you're signed in with your Google account
        
        <Note>
          Some devices may have alternative app stores. PARKINMX is distributed through Google Play Store.
        </Note>
      </Step>
      
      <Step title="Search for PARKINMX">
        1. Tap the **search bar** at the top of the screen
        2. Type **"PARKINMX"**
        3. Tap the search icon or press Enter
        4. Select the PARKINMX app from search results
        
        **App Identifier:**
        - Package: `com.alejandromv03.parkinmx`
        - Developer: alejandromv03
      </Step>
      
      <Step title="Install the Application">
        1. Tap the green **Install** button
        2. Review the permissions requested
        3. Tap **Accept** to begin download
        4. Wait for installation to complete (30-90 seconds depending on connection)
        
        The app icon will appear in your app drawer when ready.
        
        <Warning>
          The app requires approximately 50-100 MB download. Use WiFi to avoid cellular data charges.
        </Warning>
      </Step>
      
      <Step title="Grant Initial Permissions">
        When you first open PARKINMX, you'll be asked to grant permissions:
        
        **Required at installation:**
        - `USE_BIOMETRIC` - For fingerprint authentication
        - `USE_FINGERPRINT` - Legacy fingerprint support
        
        **Requested at runtime (when needed):**
        - **Camera** - When you scan a QR code
        - **Notifications** - For reservation alerts
        
        <Tip>
          You can change permissions later in Settings → Apps → PARKINMX → Permissions.
        </Tip>
      </Step>
      
      <Step title="First Launch Setup">
        1. Tap the PARKINMX icon to launch
        2. Watch the 2-second splash screen
        3. Arrive at the login screen
        4. Tap **"Regístrate"** to create your account
        
        ### Enable Biometric Authentication
        
        After first login, set up fingerprint or face unlock:
        
        1. Tap **"Enable Biometric Login"** when prompted
        2. Follow your device's biometric setup
        3. Test the biometric login by logging out and back in
        
        **Supported biometric methods:**
        - Fingerprint sensor
        - Face unlock (on supported devices)
        - Iris scanner (on Samsung devices)
      </Step>
    </Steps>

    ## Troubleshooting Android Installation
    
    <Accordion title="Play Store won't download the app">
      **Common solutions:**
      1. Clear Play Store cache: Settings → Apps → Play Store → Storage → Clear Cache
      2. Check internet connection (switch between WiFi and mobile data)
      3. Ensure sufficient storage space (Settings → Storage)
      4. Update Google Play Store to latest version
      5. Remove and re-add your Google account
      6. Restart your device
    </Accordion>
    
    <Accordion title="Installation pending or stuck">
      **Try these steps:**
      1. Open Play Store → Menu → My apps & games
      2. Find PARKINMX in the queue
      3. Tap to cancel and retry
      4. Clear Play Store data: Settings → Apps → Play Store → Storage → Clear Data
      5. Reboot device and try again
    </Accordion>
    
    <Accordion title="Fingerprint authentication not working">
      **Check settings:**
      1. Go to Settings → Security → Fingerprint
      2. Ensure fingerprint is enrolled
      3. In PARKINMX, go to Menu → Security
      4. Toggle biometric login off and on again
      5. Make sure sensor is clean and dry
    </Accordion>
    
    <Accordion title="Camera permission denied">
      **To manually grant camera access:**
      1. Open Settings → Apps → PARKINMX
      2. Tap **Permissions**
      3. Find **Camera** and tap it
      4. Select **Allow** or **Allow only while using the app**
      5. Return to PARKINMX and try again
    </Accordion>
    
    <Accordion title="App not compatible with device">
      **Possible reasons:**
      - Your Android version is below 5.0
      - Your device architecture is not supported (very rare)
      - Regional restrictions (contact support)
      
      **Solutions:**
      1. Update Android to the latest available version
      2. Check device specifications
      3. Try sideloading (advanced users only)
    </Accordion>

    ## Troubleshooting Web Access
    
    <Accordion title="Page won't load or shows blank screen">
      **Solutions:**
      1. Clear browser cache and cookies
      2. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
      3. Disable browser extensions temporarily
      4. Try incognito/private mode
      5. Update browser to latest version
      6. Try a different browser
    </Accordion>
    
    <Accordion title="Features not working as expected">
      **Check these items:**
      1. JavaScript is enabled (it must be on)
      2. No aggressive ad blockers interfering
      3. Browser is up to date
      4. Stable internet connection
      5. Console errors (F12 → Console tab)
    </Accordion>
    
    <Accordion title="Camera/QR scanner not working">
      **Common issues:**
      1. Camera permission was denied:
         - Chrome: Click the lock icon in address bar → Site Settings → Camera → Allow
         - Safari: Safari menu → Settings for This Website → Camera → Allow
      2. Camera is in use by another app
      3. Browser doesn't support getUserMedia API (very old browsers)
      4. Try HTTPS connection (camera requires secure context)
    </Accordion>
    
    <Accordion title="Can't receive notifications">
      **Enable notifications:**
      1. Check site permissions: Browser settings → Site Settings → Notifications
      2. Ensure notifications are not blocked at OS level
      3. Re-grant permission: Click the bell icon when prompted
      4. Check browser notification settings
    </Accordion>
    
    <Accordion title="Slow performance on web">
      **Optimization tips:**
      1. Close unnecessary browser tabs
      2. Clear browser cache
      3. Disable heavy browser extensions
      4. Check your internet speed
      5. Use a modern browser (Chrome/Safari/Firefox latest)
      6. Close other applications using memory
    </Accordion>

## Development Installation

For developers wanting to run PARKINMX locally:

<Accordion title="Development Setup">
  ### Prerequisites
  
  ```bash
  # Required software
  - Node.js 18+ and npm
  - Expo CLI
  - iOS: Xcode 14+ (Mac only)
  - Android: Android Studio with SDK
  ```
  
  ### Clone and Install
  
  ```bash
  # Clone the repository
  git clone https://github.com/AlejandroMV03/PARKINMX.git
  cd PARKINMX
  
  # Install dependencies
  npm install
  ```
  
  ### Configure Firebase
  
  1. Create a Firebase project
  2. Add your Firebase config
  3. Enable Authentication and Firestore
  
  ### Run Development Server
  
  ```bash
  # Start Expo dev server
  npm start
  
  # Or run on specific platform
  npm run ios      # iOS simulator
  npm run android  # Android emulator
  npm run web      # Web browser
  ```
  
  See [Development → Architecture](/development/architecture) for complete development setup.
</Accordion>
